
# Sprint 1 goals
Our goal for this sprint is to initialize our game devolvement structure. This will be achieved by implementing a basic start menu that can used to start the game and eventually change game settings. A game logo and background will also be implemented to aid in achieving a theme. To help with our structuring, we will create a UML diagram to visualize some of the classes we use over the development of *Rythmizer*.
To aid in communication, we will also setup a Discord server in which we can assign tasks efficiently to scrum members.
